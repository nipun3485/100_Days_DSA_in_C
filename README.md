# 100 Days of DSA in C

## Challenge Description
This repository contains my 100 Days of Data Structures and Algorithms (DSA) Challenge implemented using the C programming language.

I am Nipun Singh, a Semester 2 B.Tech CSE student at UPES. This challenge is aimed at building strong problem-solving skills, mastering core DSA concepts, and maintaining consistency through daily practice.

---

## Challenge Objectives
- Practice Data Structures and Algorithms daily for 100 days
- Strengthen core fundamentals of DSA using C
- Improve logical thinking and problem-solving skills
- Write clean, efficient, and well-structured C code
- Build a public learning record using GitHub
- Prepare for technical interviews and placements

---

## Tech Stack
- Programming Language: C  
- Compiler: GCC  
- Editor: Gedit  
- Version Control: Git  
- Platform: GitHub  

---

## Challenge Rules
- Solve at two DSA problem daily
- Commit code every day to maintain consistency
- Focus on understanding over memorization
- Keep solutions simple and optimized
